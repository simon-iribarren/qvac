## Overview

[Bare module](https://bare.pears.com) that adds support for translation in QVAC using either [`nmt.cpp`](https://github.com/tetherto/qvac-lib-infer-nmtcpp) or [Bergamot](https://browser.mt) as the inference engine.

## Models

You should load a model compatible with your chosen inference engine:
- `nmt.cpp` (default): OPUS-MT or IndicTrans2, converted to GGML. Model file format: `*.bin`.
- Bergamot: Bergamot model bundle. Required files: model `*.bin` + `vocab*.spm`.

## Requirement

Bare $\geq$ v1.24

## Installation

```bash
npm i @qvac/translation-nmtcpp
```

<WrapCode>
```js title="example.js"
'use strict'

const TranslationNmtcpp = require('@qvac/translation-nmtcpp')
const HyperdriveDL = require('@qvac/dl-hyperdrive')

const text =
  'La traduzione automatica ha rivoluzionato il modo in cui comunichiamo attraverso le barriere linguistiche nel mondo digitale moderno.'

async function main () {
  // 1. Create `DataLoader`
  const hdDL = new HyperdriveDL({
    // The hyperdrive key for it->en translation model weights and config
    key: 'hd://9ef58f31c20d5556722e0b58a5d262fd89801daf2e6cb28e3f21ac6e9228088f'
  })

  // 2. Create the `args` object
  const args = {
    loader: hdDL,
    params: { mode: 'full', dstLang: 'en', srcLang: 'it' },
    diskPath: './models',
    modelName: 'model.bin'
  }

  // 3. Optional: Create config object
  const config = { beamsize: 4 }

  // 4. Create Model Instance
  const model = new TranslationNmtcpp(args, config)

  // 5. Load model
  await model.load()

  try {
    // 6. Run the Model
    const response = await model.run(text)

    await response
      .onUpdate(data => {
        console.log(data)
      })
      .await()

    console.log('translation finished!')
  } finally {
    // 7. Unload the model
    await model.unload()

    // Close the DataLoader
    await hdDL.close()
  }
}

main().catch(console.error)
```
</WrapCode>

<Step>
Run `example.js`:
</Step>

<Step>
```bash
bare example.js
```
</Step>
</Steps>

## Usage
### 1. Create `DataLoader`

In QVAC, the `DataLoader` class provides an interface for fetching model weights and other resources crucial for running AI models. A `DataLoader` instance is required to successfully instantiate a translation model.

```js
const HyperdriveDL = require('@qvac/dl-hyperdrive')

const hdDL = new HyperdriveDL({
  key: 'hyperdrive-key-hex', // (Required) The Hyperdrive key containing model files.
  store: corestore // (Optional) Corestore instance (defaults to in-memory store)
})
```

### 2. Create the `args` object

The `args` object contains the `DataLoader` plus translation parameters (source/target languages) and optional stats.

```js
const args = {
  loader: hdDL, // or fsDL
  params: {
    mode: 'full',
    srcLang: 'en',
    dstLang: 'it'
  },
  opts: {
    stats: true
  }
}
```

### 3. Create the `config` object

This config controls decoding and constraints for generation.

```js
const config = {
  beamsize: 4,
  lengthpenalty: 0.6,
  maxlength: 128,
  repetitionpenalty: 1.2,
  norepeatngramsize: 2,
  temperature: 0.8,
  topk: 40,
  topp: 0.9
}
```

### 4. Create model instance

```js
const TranslationNmtcpp = require('@qvac/translation-nmtcpp')
const model = new TranslationNmtcpp(args, config)
```

### 5. Load model

```js
await model.load()
```

### 6. Run the model

```js
const response = await model.run('Hello world! Welcome to the internet of peers!')

await response
  .onUpdate(outputChunk => {
    console.log(outputChunk)
  })
  .await()

if (response.stats) {
  console.log('Translation completed in:', response.stats.totalTime, 'ms')
}
```

### 7. Unload the model

```js
await model.unload()
```
