## Overview

Each addon is a [Bare module](https://bare.pears.com) that adds an inference engine to QVAC platform, enabling one or more AI tasks.

_If you use the SDK, you **don’t need** to use addons directly._ The SDK is the canonical way to build with QVAC: it unifies all addon functionalities under a single interface, allows QVAC to run on JS runtimes beyond Bare, and provides out-of-the-box P2P connectivity.

### When to use addons?

You may choose to use addons directly **instead** of the SDK if your project meets all of the following criteria:
- It runs exclusively on Bare.
- It is already part of the Pear ecosystem **or** does not require P2P connectivity.
