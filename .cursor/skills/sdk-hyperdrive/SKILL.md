# SDK Hyperdrive

## What

API reference for Hyperdrive (distributed file system) and the qvac-lib-dl-hyperdrive download library built on top of it.

## When to Use

- Working with Hyperdrive read/write/list/download operations
- Working with HyperDriveDL (the QVAC download library)
- Debugging P2P file downloads, swarm connections, or drive replication
- Understanding Hyperdrive entry structure, blobs, or versioning
