# SDK Cache Management

## What

API reference for the SDK's model cache management: checking cache status, downloading models, loading/unloading models, and managing the model lifecycle.

## When to Use

- Working with getModelInfo, downloadAsset, loadModel, or unloadModel APIs
- Implementing model cache checks or pre-download logic
- Debugging cache states (not downloaded, cached, loaded, memory-only)
- Understanding cache directory structure or file naming
