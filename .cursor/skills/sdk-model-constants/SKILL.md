# SDK Model Constants

## What

Reference for model constants (rich metadata objects), the ModelSrcInput union type, and Zod transform schemas for model source handling.

## When to Use

- Working with model constants (WHISPER_TINY, LLAMA_3_2_1B, etc.)
- Understanding ModelConstant, ModelDescriptor, or ModelSrcInput types
- Using modelInputToSrcSchema or modelInputToNameSchema transforms
- Modifying the model code generation script (scripts/update-models.js)
- Adding new model source handling to API endpoints
