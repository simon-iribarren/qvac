# SDK Whisper Examples

## What

Example code for Whisper transcription using the qvac-lib-infer-whispercpp addon with filesystem-based model loading.

## When to Use

- Implementing or debugging Whisper transcription
- Understanding audio format conversion (PCM to float32)
- Working with TranscriptionWhispercpp model configuration
