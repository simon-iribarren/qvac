---
title: "getModelByName( )"
titleStyle: code
description: "Looks up a model in the built-in catalog by its constant name (e.g. \"LLAMA_3_2_1B_INST_Q4_0\"). Returns the matching registry item or undefined if not found."
---

```typescript
function getModelByName(name: string): RegistryItem | —
```

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| `name` | `string` | ✓ | The catalog constant name of the model (e.g. `"WHISPER_TINY"`, `"GTE_LARGE_FP16"`) |

## Returns

```typescript
RegistryItem | —
```

The matching `RegistryItem` if found, or `undefined` if no model with that name exists in the catalog.

## Example

```typescript
import { getModelByName } from "@qvac/sdk";

const model = getModelByName("LLAMA_3_2_1B_INST_Q4_0");
if (model) {
  console.log(model.registryPath, model.modelType);
}
```
