---
title: "resume( )"
titleStyle: code
description: "Resumes all suspended Hyperswarm and Corestore resources

Idempotent — calling while already active is a no-op.
Also serves as the recovery path after a partial suspend failure."
---

```typescript
function resume(): Promise
```

```typescript
Promise
```

| Error | When |

## Example

```typescript

await resume();
```
