---
title: "getModelByPath( )"
titleStyle: code
description: "Looks up a model in the built-in catalog by its registry path. Returns the matching registry item or undefined if not found."
---

```typescript
function getModelByPath(registryPath: string): RegistryItem | —
```

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| `registryPath` | `string` | ✓ | The full registry path of the model (e.g. `"llama-3.2-1b-instruct-q4_0-gguf"`) |

## Returns

```typescript
RegistryItem | —
```

The matching `RegistryItem` if found, or `undefined` if no model with that registry path exists in the catalog.

## Example

```typescript
import { getModelByPath } from "@qvac/sdk";

const model = getModelByPath("llama-3.2-1b-instruct-q4_0-gguf");
if (model) {
  console.log(model.name, model.modelType);
}
```
