```typescript
function embed(params: { modelId: string; text: string }, options?: { forceNewConnection?: boolean; healthCheckTimeout?: number; profiling?: { enabled?: boolean; includeServerBreakdown?: boolean; mode?: "summary" | "verbose" }; timeout?: number }): Promise
```

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| `params` | `\{ modelId: string; text: string \}` | ✓ | The parameters for the embedding |
| `options` | `\{ forceNewConnection?: boolean; healthCheckTimeout?: number; profiling?: \{ enabled?: boolean; includeServerBreakdown?: boolean; mode?: "summary" | "verbose" \}; timeout?: number \}` | ✗ | Optional RPC options including per-call profiling |

## Returns

```typescript
Promise
```



## Throws

| Error | When |
| --- | --- |
| `When the response type is invalid or when the embedding fails` |  |
