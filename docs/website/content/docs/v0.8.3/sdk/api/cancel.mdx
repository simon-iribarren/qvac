---
title: "cancel( )"
titleStyle: code
description: "Cancels an ongoing operation."
---

```typescript
function cancel(params: { modelId: string; operation: "inference" } | { clearCache?: boolean; delegate?: { healthCheckTimeout?: number; providerPublicKey: string; timeout?: number; topic: string }; downloadKey: string; operation: "downloadAsset" } | { operation: "rag"; workspace?: string }): Promise
```

## Parameters

| Name | Type | Required? | Description |
| --- | --- | :---: | --- |
| `params` | `\{ modelId: string; operation: "inference" \} | \{ clearCache?: boolean; delegate?: \{ healthCheckTimeout?: number; providerPublicKey: string; timeout?: number; topic: string \}; downloadKey: string; operation: "downloadAsset" \} | \{ operation: "rag"; workspace?: string \}` | ✓ | The parameters for the cancellation |

## Returns

```typescript
Promise
```



## Throws

| Error | When |
| --- | --- |
| `When the response type is invalid or when the cancellation fails` |  |

## Example

```typescript
// Cancel inference
await cancel({ operation: "inference", modelId: "model-123" });
```

```typescript
// Pause download (preserves partial file for automatic resume)
await cancel({ operation: "downloadAsset", downloadKey: "download-key" });
```

```typescript
// Cancel download completely (deletes partial file)
await cancel({ operation: "downloadAsset", downloadKey: "download-key", clearCache: true });
```

```typescript
// Cancel delegated remote download
await cancel({
  operation: "downloadAsset",
  downloadKey: "download-key",
  delegate: { topic: "topicHex", providerPublicKey: "peerHex" },
});
```

```typescript
// Cancel RAG operation on default workspace
await cancel({ operation: "rag" });
```

```typescript
// Cancel RAG operation on specific workspace
await cancel({ operation: "rag", workspace: "my-workspace" });
```
